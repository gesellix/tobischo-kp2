kp2
===

kp2 is a commandline tool for accessing Keepass 2 files (kdbx).

[gokeepasslib](https://github.com/tobischo/gokeepasslib) is used to read the file.

### Install

```
go get -u github.com/tobischo/kp2
```

### Usage

```
kp2 --help
```
